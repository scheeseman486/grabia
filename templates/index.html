    Speed Schedule

    Set bandwidth limits that activate automatically at specific times. Set 0 KB/s to pause downloads during that window. When no rule matches, downloads run uncapped unless overridden by the manual limiter in the top bar.

    Post-Download Processing

    Convert downloaded files (e.g. zipped disc images to CHD or CISO). Create profiles here, then use "Process Archive" in the archive detail view.

    External Tools

    Grabia auto-detects tools on your system PATH. Install tools via your package manager (e.g. apt install mame-tools p7zip-full unrar).

    Processing Profiles

    Profiles define how files should be converted after downloading. Assign them to archives via the "Process Archive" button.
