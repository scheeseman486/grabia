# ─── Notes on Multi-Token Parentheses ───────────────────────────────────────
# Some parenthetical groups contain comma-separated values:
#   (USA, Europe) → split into: region:usa, region:europe
#   (En, Ja)      → split into: language:english, language:japanese
#   (En, Fr, De)  → split into: language:english, language:french, language:german
#
# If a parenthetical contains a mix of regions and languages, each token is
# resolved independently. Unknown tokens become unknown:token_here.
